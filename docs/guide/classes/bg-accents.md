# Background Accents

There are few classes that can be used to add background accents to elements.

## Yellow Background Accent

**Class**

```css
.bg-accent-1 {
  background-color: var(--accent-color-1);
}
```

**Usage**

```html
<div class='bg-accent-1'>
    Yellow Background Accent
</div>
```
