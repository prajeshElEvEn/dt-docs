# Headings

There a few changes made in the default [Bootstrap headings](https://getbootstrap.com/docs/5.3/content/typography/#headings).

## h1

**Class**

```css
.heading-1,
h1 {
  font-family: var(--sdlms-font-family-poppins);
  font-size: 32px;
  font-weight: 600;
  line-height: 32px;
}
```

**Usage**

```html
<p class="heading-1">
    Heading 1
</p>
```
