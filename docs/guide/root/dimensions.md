# Dimensions

There are a few dimension values used for certain elements.

## Header Height

**Variable**

```css
var(--header-height)
```

**Value**

```css
75px
```

## Left Sidebar Width

**Variable**

```css
var(--left-sidebar-width)
```

**Value**

```css
60px
```

## Right Sidebar Width

**Variable**

```css
var(--right-sidebar-width)
```

**Value**

```css
60px
```

## Right Container Width

**Variable**

```css
var(--right-container-width)
```

**Value**

```css
350px
```
