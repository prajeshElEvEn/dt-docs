# Accents

DeepThought uses a total of 5 grey-scale to create hover effects.

:::tip
Visualize the colors on [Google's color picker](https://www.google.com/search?channel=fs&client=ubuntu&q=color+picker).
:::

```css
var(--grey-scale-1)
```
